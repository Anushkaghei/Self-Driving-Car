### Ultrasonic Sensor:
VCC -> 5V on Arduino  
GND -> GND on Arduino  
Trig -> Pin 10 on Arduino  
Echo -> Pin 9 on Arduino

### Micro Servo Motor:
VCC -> 5V on Arduino  
GND -> GND on Arduino  
Signal -> Pin 5 on Arduino

### Colour Sensor:
GND -> GND on Arduino  
OUT -> Pin 7 on Arduino  
S0 -> Pin 2 on Arduino  
S1 -> Pin 3 on Arduino  
S2 -> Pin 4 on Arduino  
S3 -> Pin 6 on Arduino
